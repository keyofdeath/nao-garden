Explication a une question
==========================

Regarde si la réponse à la question n'est pas dans la base de plantes. Si ce n'est pas le cas on regarde si ce n'est pas dans la base d'outils.
Si ce n'est toujours pas le cas, on regarde sur wikipedia si la réponse à la question de l'utilisateur y est.
